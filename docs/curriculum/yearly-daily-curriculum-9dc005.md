# 1년 교육과정 — 월별 매일 1·2교시 활동 설계

메추리 생명 프로젝트를 중심으로 국어·수학 교육과정을 1년(3월~다음해 2월) 12개월에 걸쳐 매 수업일 1교시/2교시 활동으로 구조화한 커리큘럼 플랜

---

## 기본 구조

### 입란·부화 일정 (확정)
- **4월 21일(월)**: 입란 (D-17)
- **5월 8일(목)**: 부화 예상 (D-day)
- **5월 13일(수)**: 예선 촬영 (부화 후 5일)

### 시간표 구조
- **수요일 수다모여날**: 전원 참여 (1~2교시 국어/수학 통합)
- **개별수업일** (월/화/목/금): 학생별 오는 날 (1교시 주제 활동, 2교시 개별화)
- **1교시**: 주제·계절 중심 활동 (수다모여날=전체, 개별=맞춤)
- **2교시**: 수준별 심화·연습 (관찰일기, 측정, 포트폴리오 작업)

### 사계절 구조
| 계절 | 월 | 메추리 단계 | 교육 초점 |
|:---:|:---:|:---|:---|
| **봄** | 3~5월 | 준비→입란→부화→초기 성장 | 시작, 세기, 분류, 기록 시작 |
| **여름** | 6~8월 | 성장·관찰·데이터 | 측정, 그래프, 규칙성, 일기 심화 |
| **가을** | 9~11월 | 분석·포트폴리오·산란 | 비교, 통계, 발표, 한살이 완성 |
| **겨울** | 12~2월 | 이양·회고·다음 준비 | 정리, 회고, 순환, 다음 계획 |

## DB 매핑 요약

### 활동 등록 구조

모든 월별 활동은 `activities` 테이블에 사전 등록:

```sql
-- 예시: 3월 1주 "알에 대해 알아보기"
INSERT INTO activities (
  title, season, lesson_type, subject, level_variants, iepon_unit_id
) VALUES (
  '알에 대해 알아보기',
  'spring',
  'sudamoyeo',  -- 수다모여날
  'integrated',
  '{
    "L0": {"type": "image_touch", "prompt": "알 사진을 골라요"},
    "L1": {"type": "word_trace", "prompt": "알을 따라 써요"},
    "L2": {"type": "sentence", "prompt": "알을 보고 한 문장 써요"},
    "L3-4": {"type": "report", "prompt": "알의 특징을 관찰하고 기록해요"}
  }'::jsonb,
  '국어_1학년_1단원'
);
```
