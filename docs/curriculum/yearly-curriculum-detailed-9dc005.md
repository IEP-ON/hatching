## 4. 수다모여날(수요일) 주간 활동 계획

### 범례
- **박·서** = 국어1/수학1 (박창율, 서재민)
- **신·김·조** = 국어2/수학2 (신지민, 김효주, 조사영)

## 5. 10월 본선 수업 상세 설계

> **핵심**: 전시회가 아닌 **수업**. 특수교육 개별화·수준별 교육이 드러나고, 차시 안에서 학생별 배움·성취가 뚜렷해야 함.

### 5.1 수업 개요

| 항목 | 내용 |
|:---|:---|
| **일시** | 10월 15일(수) 수다모여날 1~2교시 |
| **대상** | 전체 6명 |
| **주제** | "생명의 순환 — 우리가 기록한 이야기" |
| **핵심질문** | "봄에 만난 알이 가을에 다시 알을 낳았어요. 무엇이 달라졌고, 무엇이 같을까?" |

### 5.2 활동1 (1교시): "봄의 알 ↔ 가을의 알 — 비교하고 기록하기"

**도입 (5분)**: 4월 입란 사진 → 10월 산란 사진 나란히 보여주며 "무엇이 같고 무엇이 다를까?"

| 학생 | 단원 | 성취기준 | 활동 과제 | 기대 성취 |
|:---|:---|:---|:---|:---|
| **서재민** (L0) | 국1 2-3 그림일기 | 겪은 일을 글과 그림으로 표현 | ① 봄·가을 알 사진 2장에서 다른 점 찾기 (터치)<br>② 다른 점 1가지를 따라쓰기로 표현 | "크다/작다" 등 1단어 따라쓰기 완성 |
| **박창율** (L1) | 국1 2-3 그림일기 | 겪은 일을 글과 그림으로 표현 | ① 봄·가을 알 비교 그림 그리기<br>② "봄 알은 ○○, 가을 알은 ○○" 낱말 쓰기 | 비교 낱말 2개 이상 자기 힘으로 쓰기 |
| **김효주** (L1-2) | 국2 2-3 내용을 살펴요 | 글의 내용 파악 | ① 한살이 순서카드 5장 올바른 순서로 배열<br>② 각 단계 이름 빈칸 채우기 | 순서카드 5장 정확히 배열 + 3개 이상 이름 쓰기 |
| **신지민** (L2) | 국2 2-4 마음을 전해요 | 마음을 전하는 글쓰기 | ① 봄·가을 사진 보고 변화 2가지 문장 쓰기<br>② 메추리에게 축하 마음편지 1문장 | 비교 문장 2개 + 마음편지 1문장 완성 |
| **조사영** (L2) | 국2 2-4 마음을 전해요 | 마음을 전하는 글쓰기 | ① 한살이 순서별 변화 표 채우기<br>② "가장 좋았던 순간" 2문장 쓰기 | 표 4칸 이상 채우기 + 감상 2문장 |
| **민규원** (L3-4) | 국4 2-3 의견을 모아서 | 회의·토의 절차 | ① 한살이 데이터 요약 보고서 작성 (서론/본론/결론)<br>② 학급 친구들에게 발표 (1분) | 3문단 보고서 + 데이터 인용 1개 이상 |

### 5.3 활동2 (2교시): "숫자로 기록한 성장 — 데이터 비교"

**도입 (5분)**: PWA 대시보드에서 성장 그래프 함께 보기. "이 숫자들이 우리에게 알려주는 것은?"

| 학생 | 단원 | 성취기준 | 활동 과제 | 기대 성취 |
|:---|:---|:---|:---|:---|
| **서재민** (L0) | 수1 2-2 덧뺄셈(1) | 한 자리+한 자리 | ① 봄 알 3개 + 가을 알 2개 = ? (구체물)<br>② 큰 수/작은 수 고르기 | 5 이하 덧셈 1문제 정답 |
| **박창율** (L1) | 수1 2-2 덧뺄셈(1) | 두 자리+한 자리 (받아올림 없음) | ① 4월 몸무게 8g → 10월 몸무게 25g, 증가량 계산<br>② 먹이 12g + 물 5g = ? | 받아올림 없는 덧셈 2문제 정답 |
| **김효주** (L1-2) | 수2 2-3 길이 재기 | mm 단위 측정 | ① 봄 알 길이(mm)와 가을 알 길이(mm) 자로 측정<br>② 두 길이 비교하여 "○mm 더 길다" | 자로 측정 + mm 단위 비교 문장 |
| **신지민** (L2) | 수2 2-3 길이 재기 | mm 단위 측정·비교 | ① 개체별 발 크기 측정(mm)<br>② 가장 큰 발/가장 작은 발 비교, 차이 계산 | 3개체 이상 측정 + 차이 계산 |
| **조사영** (L2) | 수2 2-3 길이 재기 | mm·cm 혼합 | ① 봄~가을 키 변화 표 완성 (cm, mm)<br>② "가장 많이 자란 달"을 찾고 이유 쓰기 | 표 완성 + 최댓값 찾기 |
| **민규원** (L3-4) | 수4 2-3 소수의 덧뺄셈 | 소수 크기 비교·연산 | ① 몸무게 데이터를 소수로 변환 (25g→0.025kg)<br>② 봄·가을 몸무게 차이 소수 계산<br>③ 결과를 문장으로 해석 | 소수 변환 + 뺄셈 + 해석 문장 |

### 5.4 수업 흐름 (심사 포인트)

```
[도입 5분] 봄 알 ↔ 가을 알 사진 비교 → 핵심 질문 제시
    ↓
[활동1 30분] 수준별 비교·기록 과제
    - 교사: 3개 그룹 순회 (서+박 / 김+신+조 / 민)
    - PWA: 수준별 UI 자동 분기 → 학생 태블릿에 맞춤 과제 노출
    - 실시간 제출 → 교사 대시보드에서 진행 상태 확인
    ↓
[전환 5분] 활동1 결과 공유 (각 학생 1문장씩 발표)
    ↓
[활동2 30분] 수준별 데이터 비교 과제
    - 구체물(저울, 자) + PWA 데이터 연계
    - 민규원: 발표 리더 역할
    ↓
[마무리 10분] "오늘 배운 것" 한 문장씩 → PWA에 기록
```

**심사 시 드러나는 포인트**:
1. **개별화**: 동일 주제(알 비교)지만 6명 모두 다른 과제 → 다른 성취기준
2. **수준별 성취**: 서재민(따라쓰기 1단어) ~ 민규원(보고서 3문단)까지 명확한 차등
3. **에듀테크 통합**: PWA가 수준별 UI 자동 분기, 실시간 진행 추적
4. **데이터 기반 교육**: 실제 측정 데이터를 수학 수업에 직접 활용
5. **교육과정 연계**: 각 학생의 현재 단원·성취기준에 정확히 대응

| **센서** | 온도+습도+무게+물높이 | **온도+습도** 중심 (간소화) |
| **카메라** | 선택 사항 | **타임랩스 카메라** 추가 (부화 과정 촬영) |
| **범위** | 부화기+사육장 | 부화기 중심 (사육장은 수동) |

### IoT 데이터 흐름
```
[전란틀 구매품] → 자동 전란 (모터 내장)
[Raspberry Pi] → DHT22(온습도) + Pi Camera(타임랩스)
    → Supabase Edge Function → iot_readings / iot_images
    → PWA 대시보드 실시간 표시
```
