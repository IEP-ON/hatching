메추리 한살이를 국어·수학 통합의 중심 소재로 삼아, L0~L3-4 수준별 학습 활동 3대 트랙, 마음편지 관찰일기(GPT 전 수준 지원), 부화기 만들기 통합 에피소드를 기존 플랜(`hatching-pwa-webapp-plan-6544cf.md`)에 추가한다.

## 1. 교육 철학 — 왜 한살이인가

- **생명의 순환**: 알 → 부화 → 성장 → 산란 → 다시 알. 특수교육대상학생에게 "시작과 끝이 다시 시작되는" 경험은 L0부터 L3-4까지 모두 참여 가능한 보편적 감동.
- **교과 통합의 자연스러운 연결**
  - 수학: 세기 → 분류 → 가르기·모으기 → 측정 → 규칙성 → 데이터
  - 국어: 이름붙이기 → 관찰 표현 → 마음편지 일기 → 발표 → 보고서
- **부화기 만들기 자체가 수업**: 재활용 재료 분류(수학), 따뜻함 탐색(과학 준비), 제작 설명(국어)
- **관찰일기(마음편지)**: 생명체에게 편지 쓰듯 주기적으로 기록. 낮은 수준은 GPT 초안을 터치로 완성.

## 2. 수준별 교육 기대치

| 수준 | 학생 | 국어 목표 | 수학 목표 |
|:---:|:---|:---|:---|
| **L0** | 서재민 | AAC·그림으로 관찰 표현, GPT 일기 카드 선택·완성 | 알 갯수 세기(~5), 색상별 분류, 2개 가르기 |
| **L1** | 박창율 | 낱말 쓰기, 그림+낱말 일기 | 10 이하 가르기·모으기, 기본 분류, 단위 없는 저울 읽기 |
| **L1-2** | 김효주 | 문장 빈칸 채우기, 따라쓰기 일기 | 분류하기(속성 2가지), 같음·다름, 패턴 이어하기 |
| **L2** | 신지민·조사영 | 1~2문장 관찰 일기, 감정 표현 | 측정(g 단위), 규칙성 찾기, 날짜·기간 셈 |
| **L3-4** | 민규원 | 한살이 관찰 보고서, 발표 | 꺾은선그래프 해석, 규칙 예측, 일수 계산·비교 |

## 3. 학습 활동 3대 트랙

### 트랙 A — 수준별 수학 활동

| 순서 | 시기 | 소재 | L0 | L1~L1-2 | L2 | L3-4 |
|:---:|:---:|:---|:---|:---|:---|:---|
| 1 | 3월(봄) | 알 세기·색상 분류 | 그림 터치 세기(~5) | 숫자쓰기+블록세기 | 직접 세기+표 | 표+이유 |
| 2 | 3~4월(봄) | 가르기·모으기 | 2개 이하 조작 | 5 이하 가르기 | 10 이하 | 문제 만들기 |
| 3 | 5~6월(봄→여름) | 메추리 몸무게 측정 | 무거운/가벼운 선택 | 저울 읽기(단위 없이) | g 단위 읽기 | 꺾은선그래프 |
| 4 | 6~8월(여름) | 털 색깔 패턴 | 반복 패턴 따라하기 | 패턴 이어그리기 | 패턴 규칙 설명 | 규칙 예측+기록 |
| 5 | 9~11월(가을) | 한살이 순서 수 | 순서 카드 놓기 | 1~5 순서 나열 | 날짜·기간 계산 | 일수 계산+비교 |

### 트랙 B — 수준별 국어 활동

| 순서 | 활동 | L0 | L1~L1-2 | L2 | L3-4 |
|:---:|:---|:---|:---|:---|:---|
| 1 | 알 이름 붙이기 | 그림카드 3개 중 선택 | 낱말 따라쓰기 | 이름+이유 1문장 | 이름+이유 2~3문장 |
| 2 | 마음편지 관찰일기 | GPT 초안 → 터치 선택 | GPT 초안 → 낱말 완성 | GPT 초안 → 문장 수정 | 자유 작성+GPT 검토 |
| 3 | 부화기 만들기 설명 | 재료 그림 고르기 | 재료 이름 쓰기 | 재료+이유 문장 | 설계도 글쓰기 |
| 4 | 성장 발표 | 사진 보고 말하기 | 낱말 카드로 발표 | 1~2문장 발표 | 보고서 발표 |

### 트랙 C — 부화기 만들기 프로젝트 (통합 에피소드)

**시기**: 3월 2~3주차 (수다모여날 + 개별수업 연계)

**활동 흐름** (1수업 = 수다모여날 활동 1회, 이후 개별 심화):
1. **재활용 재료 가져오기 → 분류하기** (수학: 분류)
2. **어떤 재료가 따뜻한지 탐색·예측** (과학적 사고 준비)
3. **부화기 설계 및 제작** (실과 준비, 만들기)
4. **만든 부화기 소개 글쓰기** (국어 쓰기)
5. **온도 비교 측정** (수학: 측정, IoT 연동 가능)

**level_variants 예시** (`activities.level_variants` jsonb):
```json
{
  "L0":   { "type": "image_select",     "prompt": "따뜻한 재료를 골라요",              "items": ["솜", "신문지", "유리"] },
  "L1":   { "type": "sort_and_word",    "prompt": "재료를 종류별로 나누고 이름을 써요" },
  "L2":   { "type": "design_and_sentence", "prompt": "부화기를 그리고 재료와 이유를 써요" },
  "L3-4": { "type": "report",           "prompt": "설계 이유와 온도 측정 결과를 써요" }
}
```

## 4. 마음편지 관찰일기 — GPT API 통합

### 개념
- 폴라로이드처럼 사진을 찍고 생명체에게 편지 쓰듯 관찰 일기를 남기는 활동
- **모든 수준**에서 GPT 초안 생성 옵션 제공, 수준별 출력 품질 분기
- 주기: 주 1~2회 (수다모여날 이후 + 개별 수업 시)

### 수준별 GPT 생성 전략

| 수준 | 입력 | GPT 출력 | 학생 조작 |
|:---:|:---|:---|:---|
| **L0** | 사진 + 날짜 | 1~2문장, 쉬운 어휘 ("오늘 알이 깨졌어요. 반가워요!") | 그림카드 감정 선택 후 저장 |
| **L1** | 사진 + 낱말 2~3개 선택 | "오늘 ___ 를 봤어요. ___ 했어요." 빈칸 문장 | 빈칸 터치 입력 |
| **L1-2** | 사진 + 짧은 관찰 한 줄 | 문장 2~3개 초안 | 문장 수정·선택 |
| **L2** | 사진 + 관찰 2~3문장 | 정리된 일기 초안 (4~5문장) | 직접 수정 |
| **L3-4** | 사진 + 자유 관찰 | 문체·어휘 제안 + 오탈자 교정 | 자유 편집 |

### API 설계

```
POST /api/diary/generate
  요청: { photo_url, student_level, keywords[], raw_text? }
  응답: { draft_text, suggestions[] }
  모델: gpt-4o-mini
  프롬프트: 수준 코드(L0~L3-4)별 시스템 프롬프트 분기
  보안: 학생 이름(PII) 전송 금지 — 수준 코드와 관찰 키워드만 전달
```

### DB 변경 (마이그레이션 추가)

기존 `observations` 테이블에 컬럼 추가:

| 컬럼 | 타입 | 설명 |
|:---|:---|:---|
| `diary_type` | text | `heart_letter` / `general` |
| `gpt_draft` | text | GPT 생성 원본 초안 (보존용) |
| `gpt_used` | boolean | GPT 사용 여부 |

> `media_urls`(기존)로 사진 URL 저장 — 변경 없음
