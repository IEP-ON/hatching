## 📌 주요 개념

### 수준 코드 (Level Code)
- **L0**: 유치원 6~7세 도전 수준 (AAC, 그림 중심)
- **L1**: 1학년 교육과정
- **L1-2**: 1~2학년 경계
- **L2**: 2학년 교육과정
- **L3-4**: 3~4학년 사이

### 사계절 내러티브
- **봄**: 생명의 시작, 만남과 설렘
- **여름**: 함께 자라는 여름, 돌봄과 기록
- **가을**: 데이터로 읽는 성장 이야기
- **겨울**: 작별과 감사, 다음 봄을 기다리며

### 학습 활동 3대 트랙
1. **수다모여날** (수요일 통합 수업)
2. **개별 수업** (월/금 수준별 맞춤)
3. **마음편지 관찰일기** (GPT 지원 일기 작성)
